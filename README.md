# 🐶 TinDog - Dog Dating Website

A fun and responsive landing page for a fictional dog dating app called **TinDog**. This project is built using **HTML, CSS, and Bootstrap 5**, focusing on layout design and UI components.

---

## 🚀 Live Demo

https://bhushanmalviya02.github.io/tindog/

---

## 📌 Features

* 📱 Fully responsive design using Bootstrap
* 🐕 Clean and modern UI for a pet-themed app
* 💳 Pricing section with different plans
* ⭐ Testimonials section
* 🔥 Reusable components and structured layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5
